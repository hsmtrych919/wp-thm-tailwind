# クラス名変更ログ

> SCSS → Tailwind CSS 移行に伴うクラス名変更の全記録
> PHP テンプレート修正時のリファレンスとして使用
> **各 Phase の作業中にリアルタイムで更新すること**

---

## 利用方法

- **旧クラス名**で検索 → 対応する**新クラス名**を確認
- 一括置換スクリプトの入力データとしても使用可能
- ステータス: ✅ 置換済み / ⏳ 未着手
