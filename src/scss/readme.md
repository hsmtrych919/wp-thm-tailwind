# css単位の使い分け

## rem
font-size,margin,padding など基本単位で使用
16pxベースのrem換算 @function作成して作成

###  ユーティリティクラス .mt__

.mb__, pt__, .pb__ は廃止
.mt__ に統一

.mt__ のスタイル2種類

- .mt__#{$name}#{$infix}
ブレークポイントごとにレスポンシブにクラス作成
$space-values: (
  0_5: 4,0_75: 6,1: 8,1_25: 10,1_5: 12,2: 16,2_25: 18,2_5: 20,3: 24,3_5: 28,
  4: 32,4_5: 36,5: 40,6: 48,7: 56,8: 64,9: 72,10: 80,12: 96,
  13: 104,14: 112,15: 120,16: 128,18: 144,20: 160,
) !default;

- .mt__#{$name}
clamp を使って変動。pc時のサイズでセット





## vw
ブレークポイントごとに変数にvwをセット
:root {
  --unit: #{g.get_vw(1, 390)};
  --gutter: #{g.get_vw(6, 390)};
  --gutter-row: #{g.get_vw(16, 390)};
  以降ブレークポイントで更新
}

横方向の余白は vwを換算した変数利用



## px
レイアウトの大枠などサイズを厳密に決めたい場合
